WDAT

Hello and welcome to the Web Developer Accessibility Test

The goal of this test is to assess wether a web developer truly understands and can implement the changes needed in a website to make it as accessible as possilbe

In the test you will find 50 multiple choice questions and a place where the user can input any feedback they wish to share with the purpose of making a better test

This test aims to cover common accessibility pitfalls and real life user struggles

As content changes so to this test should change to be as current as possible
